โครงการพัฒนาระบบเทคโนโลยีเพื่อสังคม

CareConnect:
ระบบจองผู้ดูแลผู้สูงอายุ
ถึงบ้านแบบครบวงจร

เชื่อมโยงครอบครัวและผู้ดูแลมืออาชีพด้วยระบบดิจิทัลที่ปลอดภัย

ที่มาและความสำคัญ

ประเทศไทยกำลังเข้าสู่ "สังคมผู้สูงอายุ" อย่างเต็มตัว ทำให้ความต้องการผู้ดูแลเพิ่มขึ้นมหาศาล

  • หาผู้ดูแลที่ไว้ใจได้ยาก
  • ตารางเวลาไม่ชัดเจน
  • ขาดมาตรฐานการชำระเงินและความปลอดภัย

ทางออกของเรา

ระบบตัวกลางที่คัดกรองผู้ดูแล จัดระเบียบตารางงาน และรองรับการชำระเงินที่ตรวจสอบได้

วัตถุประสงค์โครงการ

ความสะดวก

ให้ผู้ใช้จองบริการได้ในไม่กี่คลิก พร้อมระบบชำระเงินในตัว

ประสิทธิภาพ

ช่วยผู้ดูแลจัดการคิวงานผ่านหน้าต่าง Admin ได้อย่างมีระเบียบ

มิตรต่อผู้ใช้

ออกแบบหน้าจอ (UI) ที่เรียบง่าย เหมาะกับคนทุกวัย

ฟังก์ชันหลักของระบบ

  • ● ระบบสมาชิก: เข้าใช้งานได้ทั้งฝั่งผู้ใช้และผู้ดูแลระบบ
  • ● การเลือกบริการ: ครอบคลุมตั้งแต่งานพื้นฐานจนถึงงานเฉพาะทาง
  • ● ระบบที่อยู่และชำระเงิน: ระบุพิกัดบ้านและจ่ายเงินล่วงหน้า
  • ● Admin Dashboard: ตรวจสอบคำขอและกดอนุมัติงาน

รายละเอียดทางเทคนิค

วัตถุประสงค์ของเว็บไซต์

เพื่อเป็นแพลตฟอร์มกลางในการจัดหาและจองบริการผู้ดูแลผู้สูงอายุที่มีคุณภาพ เข้าถึงง่าย และมีความปลอดภัยสูงสำหรับสังคมไทยยุคดิจิทัล

เทคโนโลยีที่ใช้พัฒนา

HTML5 CSS3 (Tailwind) JavaScript Node.js Firebase/Firestore

สถาปัตยกรรมแบบ Modern Web App รองรับการแสดงผลแบบ Responsive บนทุกอุปกรณ์
